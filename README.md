# Malware Analysis using Wireshark

<h2>Project description</h2>
The goal of this project is to demonstrate the efficient analysis of a PCAP file and conduct static malware analysis.


<h2>Utilities Used</h2>

- <b>VirtualBox</b>
- <b>Wireshark</b> 
- <b>VirusTotal</b>
- <b>AbuseIPDB</b>
